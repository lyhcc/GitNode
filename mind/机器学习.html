  期望的性质

   无条件

    E(kX) = kE(X)

    E(X+Y) = E(X)+E()

   X,Y相互独立

    E(XY) = E(X)E(Y)

    反之，X,Y不一定相互独立，只能说明它们不相关

  方差

   Var(X) = E{X -[E(X)]^2}

  方差性质

   无条件

    Var(c) = 0

    Var(X + c) = Var(X)

    Var(kX) = k^2 * Var(X)

   X，Y相互独立

    Var(X+Y) = Var(X) + Var(Y)

  协方差

   Cov(X,Y) = E{[X - E(X)][Y-E(Y)]}

  协方差性质

   Cov(X,Y) = Cov(Y,X)

   Cov(aX+b , cY+d) = acCov(X,Y)

   Cov(X1+X2, Y) = Cov(X1, Y) + Cov(X2, Y)

   Cov(X , Y) = E(XY) - E(X)E(Y)

  协方差和独立、不相关

   X,Y独立时， E(XY) = E(X)(Y)

   X，Y独立时，Cov(X, Y) = 0

   但是，Cov（X,Y）= 0 称X,Y不相关

  协方差的意义

   等于0

    不相关（是指线性独立）

   大于0

    变化趋势相同

   小于0

    变化趋势相反

  协方差的上界

   |Cov(X,Y)| < σ1σ2

    当且仅当X和Y线性相关是等号成立
